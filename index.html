Different types of selectors New

1- Element Selectors:

The element selector allows developers to select HTML elements based on their element type.

2- ID Selectors:

The ID selector uses the id attribute of an HTML.
Since the id is unique within a webpage, it allows the developer to select a specific element for styling.
ID selectors are prefixed with a # character.

3- Class Selectors:

Elements can also be selected based on the class attribute applied to them.
The CSS rule has been applied to all elements with the specified class name.
The class selector is prefixed with a . character.

6- Child Selectors ( Important ) !!!

Child selectors are more specific than descendant selectors.
They only select elements that are immediate descendants (children) of a selector (the parent).
